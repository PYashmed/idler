# 任务栏时钟显示秒

## 修改windows注册表

+ **`win+r`** 打开windows注册表编辑器
+ 地址栏中键入 **`计算机\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced`**
+ 右键新建 **`DWORD(32位)值`** 重命名为 **`ShowSecondsInSystemClock`**
+ 双击修改其值为 **`1`** **`16进制`**

## 重启windows资源管理器（任务栏）

+ 打开任务管理器
+ **`windows进程--windows资源管理器--选中--右下角重启进程`**
